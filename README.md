## Objetivo
<p>Essa aplicação consiste em um cadastro de usuário web onde se pode: cadastrar, logar, visualizar e alterar os dados da base cadastral.</p>

## Tecnologias 
##### • HTML  
##### • CSS3 
##### • PYTHON 
##### • FLASK 
##### • PHPMYADMIN
##### • FLASK_MYSQLDB
##### • WAMPSERVER
