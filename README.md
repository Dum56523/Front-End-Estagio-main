## Objetivo
<p>Essa aplicação consiste em um cadastro de usuário web onde ele possa se cadastrar,logar,ver seus dados e alterar seus dados.</p>

## Tecnologias 
##### HTML  
##### CSS3 
##### PYTHON 
##### FLASK 
##### PHPMYADMIN
##### FLASK_MYSQLDB
##### WAMPSERVER
